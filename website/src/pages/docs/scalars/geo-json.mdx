# GeoJSON

A GeoJSON object as defined by [RFC 7946](https://datatracker.ietf.org/doc/html/rfc7946).

## Usage with Codegen

In addition to adding `GeoJSON: GeoJSONResolver.extensions.codegenScalarType` to the `scalars`
mapping in your Codegen configuration, include the import for the `GeoJSONObject` type using the
`add` plugin.

Hereâ€™s an example of how your `CodegenConfig` should look:

```typescript
{
  ...,
  generates: {
    "<your-generated-file-path>": {
      config: {
        ...,
        scalars: {
          GeoJSON: GeoJSONResolver.extensions.codegenScalarType,
        },
      },
      plugins: [
        ...,
        {
          add: {
            content: [
              "import type { GeoJSONObject } from 'graphql-scalars/dist/typings/scalars/GeoJSON/types';",
            ],
          },
        },
      ],
    },
  }
};
```

This ensures that the required `GeoJSONObject` type is imported into your generated code.

## Accessing the JSON Schema

If you require the full JSON schema, it is not accessible via `.extensions.jsonSchema` (as is the
case for most scalars). Instead, the schema can be manually imported when needed.

### ESM Import:

```ts
import type { geojsonSchema } from 'graphql-scalars/dist/esm/scalars/GeoJSON/jsonSchema'
```

### CommonJS Import:

```ts
import type { geojsonSchema } from 'graphql-scalars/dist/cjs/scalars/GeoJSON/jsonSchema'
```
